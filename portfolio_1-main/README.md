## 🚀 LOCAL TRANSMISSION (SETUP)

To run the Forge on your local machine:

1. **Clone the repository**
   ```bash
    git clone https://github.com/Utkarsh322/portfolio_1.git
   ```

2. **Initialize dependencies**
   ```bash
   npm install
   ```

3. **Ignite the server**
   ```bash
   npm run dev
   ```
